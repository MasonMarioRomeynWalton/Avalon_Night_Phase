To copy the repository:
```
git clone https://github.com/MasonMarioRomeynWalton/Avalon_Night_Phase
```

To set up and run the following commands:
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
This will set up a virtual environment and install the dependencies in "requirements.txt". After this you should be able to host the server with
```
python main.py
```
